## 5. Scope Check (Mandatory)

Before promoting any change, verify:

- Does this strengthen the core task of technical co-writing?
- Does this improve author reasoning rather than merely adding convenience?
- Does this avoid turning the skill into a generic assistant?
- Is this stable enough to become a default rule?
- Should this remain external instead of entering the core skill?

Decision:

- Reject
- Keep as local preference
- Revisit after more usage
- Promote to core skill
